# Interface

A data integration and dashboard app built with Next.js, TypeScript, and Tailwind CSS.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3002](http://localhost:3002) in your browser.

## Tech Stack

- Next.js 15 (App Router, Turbopack)
- React 19
- TypeScript
- Tailwind CSS
- shadcn/ui + Radix UI
- Recharts
- Framer Motion
